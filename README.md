# PinchImageView

## 功能

1. 双击放大，双击缩小
2. 手势放大，手势缩小，图片更加跟手，超出边界会回弹
3. 滑动惯性
4. 不同分辨率图片无缝切换
5. 只有一个类，使用简单，可与ImageView替换
6. 易于与ViewPager结合（ViewPager需要做少量改动）
7. 用SwitcherImageView代替与ViewPager结合进行多图切换
